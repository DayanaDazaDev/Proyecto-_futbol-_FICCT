# 🏆 FICCT - Gestión de Campeonatos de Fútbol

**Sistema web para la administración de torneos deportivos universitarios**

## 🚀 Características

- ✅ Creación de campeonatos con múltiples formatos (Liga, Eliminación, Grupos)
- ✅ Gestión de equipos y jugadores
- ✅ Fixture automático y resultados en tiempo real
- ✅ Tabla de posiciones calculada automáticamente
- ✅ Vista pública para compartir por link
- ✅ Autenticación segura por email/contraseña

## 🛠️ Tecnologías

- **Frontend:** Next.js 14 + TypeScript + Tailwind CSS
- **Backend:** Supabase (PostgreSQL + Auth + RLS)
- **Hosting:** Vercel
